# 너굴마켓

## 🦝 프로젝트 소개
"너굴마켓"은 "너(You)"와 "굴처럼 쌓여 있는 물건들"의 만남에서 영감을 받아 만들어진 중고거래 플랫폼입니다.

사용자들이 더 이상 사용하지 않는 물건들을 쉽고 간편하게 사고팔 수 있는 공간을 제공합니다.

[너굴마켓 이용하러 가기](http://3.35.214.27:8080/)

## 📆 개발 기간
2024.11.26 - 2024.12.23
